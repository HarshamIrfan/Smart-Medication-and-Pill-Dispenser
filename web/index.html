About This Dashboard

This dashboard is the software companion to an undergraduate embedded systems mini project — a Smart Medication Dispenser & Adherence Tracker built on the ESP32 platform.

MedTrack is a browser-based medication management dashboard built to accompany a smart medication dispenser prototype developed as an academic project.

This interface demonstrates how a companion web application can support medication scheduling, adherence tracking, data visualisation, and historical records beyond the capabilities of a constrained embedded display.

No live hardware is connected. All data is stored locally in your browser via localStorage. Nothing is transmitted to any server, and clearing browser storage will reset the application.
